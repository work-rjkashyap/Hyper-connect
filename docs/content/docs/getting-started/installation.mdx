---

Hyper Connect is available for macOS, Windows, and Linux. You can download the latest version from our [GitHub Releases](https://github.com/work-rjkashyap/Hyper-connect/releases/latest) page.

## Platform Requirements

### macOS

- macOS 10.15 (Catalina) or later.
- Apple Silicon and Intel Macs are both supported.

### Windows

- Windows 10 or later.
- 64-bit architecture.

### Linux

- Any modern distribution supporting AppImage.

## Installation Steps

<Steps>
<Step>
### Download the Installer
Go to the releases page and download the appropriate file for your OS:
- `.dmg` for macOS
- `.exe` for Windows
- `.AppImage` for Linux
</Step>

<Step>
  ### Run the Installer - **macOS**: Open the DMG and drag Hyper Connect to your Applications
  folder. - **Windows**: Run the `.exe` and follow the on-screen instructions. - **Linux**: Make the
  `.AppImage` executable (`chmod +x`) and run it.
</Step>

<Step>
### Launch the App
Open Hyper Connect. On the first run, you may need to grant permissions for local network access and firewall exceptions.
</Step>
</Steps>

<Callout type="warn" title="macOS Security">
  If you see a message saying the app is damaged or cannot be opened, please refer to the
  [Troubleshooting](../guides/troubleshooting) guide.
</Callout>
